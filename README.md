# java-cli-maven-surefire-spring-findbugs-mockito-serenity-cucumber-junit5-car-data

## Description
A POC for maven app using junit5
mockito, serenity, and cucumber framework
 with findbugs
and urefire plugin.

## Tech stack
- java
- maven
	- spring
  - junit5
  - findbugs
  - surefire
  - cucumber
	- serenity
	- mockito


## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
